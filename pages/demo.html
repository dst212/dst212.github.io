A tooltip is exactly this: a small pop-up which appears when the cursors hovers on the related text, which generally explains what the text refers, what it is or what it does.

What about tooltips' titles
The title is the label above this text. It can be set by applying the title property to the .tooltip element.
If nothing is set, a default title will show up.
